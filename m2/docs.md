## Data Source
M2 uses the IGDB API (igdb.com) to fetch Epic Games Store titles.
IGDB is an industry-standard game database owned by Twitch/Amazon,
used by major gaming platforms. It currently tracks 1623+ Epic Games
Store titles via their external_games catalog.

Epic Games does not provide a public catalog API equivalent to Steam's
IStoreService, so IGDB is the most complete legitimate data source available.
